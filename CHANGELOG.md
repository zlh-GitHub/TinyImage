# Changelog

## [1.2.0] - 2026-04-09

### 新增

- **Finder 快速操作支持键盘快捷键**：安装脚本新增快捷键配置步骤，默认注册 `⌘⇧M`，在 Finder 选中文件后直接触发压缩，无需右键菜单
- **压缩进度终端窗口**：压缩时弹出新 Terminal 窗口，实时显示每张图片的压缩比、原始大小与压缩后大小；压缩完成后按回车自动关闭窗口

### 修复

- 修复 `VOLTA_BYPASS=1` 导致终端子进程无法定位 `tiny` 命令的问题（外层脚本解析路径后作为参数传入）
- 修复 macOS BSD `mktemp` 不支持带扩展名模板导致临时脚本路径为空的问题

---

## [1.1.0] - 2026-04-08

### 新增

- **macOS Finder 快速操作**：新增 `install-macos-service.sh` 安装脚本，在 Finder 中右键图片或文件夹即可一键压缩，无需打开终端
  - 压缩在后台静默执行
  - 通过 macOS 通知中心推送开始与完成状态（含成功/跳过/失败统计）
- **CLI 直接压缩模式**：新增 `tiny compress <path>` 子命令，传入路径直接压缩，跳过交互菜单，压缩完成后自动退出，适合脚本调用

### 变更

- CLI 入口改用 `commander` 统一管理路由：`tiny`（无参数）进入交互菜单，`tiny compress <path>` 进入直接压缩模式

---
