# Changelog

## [1.1.0] - 2026-04-08

### 新增

- **macOS Finder 快速操作**：新增 `install-macos-service.sh` 安装脚本，在 Finder 中右键图片或文件夹即可一键压缩，无需打开终端
  - 压缩在后台静默执行
  - 通过 macOS 通知中心推送开始与完成状态（含成功/跳过/失败统计）
- **CLI 直接压缩模式**：新增 `tiny compress <path>` 子命令，传入路径直接压缩，跳过交互菜单，压缩完成后自动退出，适合脚本调用

### 变更

- CLI 入口改用 `commander` 统一管理路由：`tiny`（无参数）进入交互菜单，`tiny compress <path>` 进入直接压缩模式
